Este projeto é basicamente uma versão do Campo Minado que eu desenvolvi em Kotlin.

O jogo funciona com um tabuleiro em forma de matriz, onde cada célula pode ter uma mina, um número ou estar vazia, e também guarda se já foi revelada ou não.

Quando o jogo começa, as minas são colocadas aleatoriamente e o programa calcula automaticamente quantas minas existem à volta de cada posição.

O jogador vai introduzindo coordenadas para se mover no tabuleiro, e eu implementei várias validações para garantir que os movimentos são válidos e que o jogo não dá erros.

Se o jogador calhar numa mina, perde. Se conseguir chegar ao ponto final, ganha.

Também adicionei algumas funcionalidades extra, como carregar jogos a partir de ficheiros e mostrar ou esconder a legenda.

No geral, este projeto ajudou-me bastante a melhorar a minha lógica de programação e a forma como organizo o código.
